# 2026-02-08 - 사장님 피드백

## 중요한 피드백 (사장님)

### 작업 방식 개선 필요
1. **.env 위치**: workspace가 아니라 `workspace/github/ai-lounge/`에 있음을 확인해야 함
2. **kubectl 사용법**: memory에 kubectl 사용법과 .env 위치가 다 나오는데 활용하지 않음
3. **배포 방식**: git commit 하면 자동으로 argocd 반영됨. kubectl은 pod 이슈 파악 용도이지, 명령어로 배포하라고 한 게 아님
4. **팀장 역할**: 팀원이 헛소리(잘못된 보고)하면 알아서 컷하고 수정하라고 시켜야 함

### nabijiyo 서비스 관련
1. **endpoint**: .env에 있음. `ENDPOINT=mohae.uk`
   - a. subdomain: `nabijiyo.mohae.uk`를 내부 어디 url:port랑 연결해야 하는지 알아야 함
   - b. 클라우드플레어는 사장님이 직접 등록할 예정
2. **postgres**: 나(pod)가 직접 올리면 됨

### ArgoCD 오류
```
Failed to load target state: failed to generate manifest for source 1 of 1: rpc error: code = Unknown desc = Manifest generation error (cached): `kustomize build <path to cached source>/app-of-apps` failed exit status 1: Error: accumulating resources: accumulation err='accumulating resources from 'overlays/nabijiyo': '<path to cached source>/app-of-apps/overlays/nabijiyo' must resolve to a file': recursed merging from path '<path to cached source>/app-of-apps/overlays/nabijiyo': may not add resource with an already registered id: Secret.v1.[noGrp]/regcred.ai-lounge
```

**문제**: Secret.v1.regcred.ai-lounge가 중복 등록됨
**해결**: base/overlays에서 Secret 충돌 제거 필요
