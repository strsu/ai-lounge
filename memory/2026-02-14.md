## 프로젝트 상태

### 기술 스택
- **프론트엔드:** Next.js 14.x (App Router), TypeScript, Tailwind CSS
- **백엔드:** Next.js API Routes, Prisma ORM, PostgreSQL
- **데이터 수집:** Cheerio, Axios (네이버 스크래핑)
- **배포:** Docker, Kubernetes, ArgoCD (GitOps)
- **인프라:** Cloudflare Tunnel (외부 접속), PostgreSQL (데이터베이스)

### 구현된 기능 (P0 - API 완료)
✅ **맛집 검색 API** (`/api/search`)
- 데이터베이스 검색 (이름, 주소)
- 결과 제한 기능
- 리뷰, 포스팅, 요약 포함

✅ **스크래핑 API** (`/api/scrape`)
